# 磨棱 MoLeng

**把经历磨出棱角**

每天随手记，AI 帮你找出值得复盘的事，你自己写下判断和教训。时间越长，你的决策手册越厚。

---

## 这是什么

磨棱是一个 AI 辅助的个人认知提纯系统。

你只管记录，AI 帮你从中发现值得思考的事并给出参考建议，你自己写复盘和心得，系统帮你结构化存储。最终你会积累一本属于自己的决策手册——所有成功的经验、失败的教训、可复用的 SOP，随时可查、可筛选、可溯源。

**与笔记工具的区别：** 笔记工具帮你记住，磨棱帮你变强。Notion、Obsidian、思源解决的是"怎么更好地存和找"，磨棱解决的是"怎么不在同一个地方栽倒两次"。

---

## 核心流程

```
随手记（碎片）──┐
                ├──→ AI 整合成当天日记 → 你确认
写完整日记 ────┘
                                         │
                                         ▼
                              AI 提取事件 + 给出参考建议
                                         │
                                         ▼
                              你自己写复盘（打标签：成功/失败/认知迭代）
                                         │
                                         ▼
                              你自己写心得（知 = 道理，行 = SOP）
                                         │
                                         ▼
                                      入库 ✓
```

**输入端零门槛，输出端你把关。每条入库记录都是你亲手写的。**

## 功能

- **随手记** — 想到什么就记一句，碎片自动进收集箱，AI 帮你整合成当天日记
- **AI 辅助复盘** — AI 从日记中提取值得复盘的事件并给出参考建议，你看着建议自己写
- **心得提炼（知 + 行）** — 每条复盘提炼出"知"（道理）和"行"（SOP），AI 给建议你来写
- **水事件过滤** — 日记里没有决策和偏差的流水账，AI 不会硬凑复盘
- **标签筛选** — 一键筛选所有成功/失败/认知迭代的复盘
- **三层溯源** — 从任意一条心得追溯到复盘和原始日记，反过来也可以
- **认知迭代链** — 对同一个问题的认知更新时，新旧心得自动关联，形成版本链
- **数据导出** — JSON / CSV 导出，数据永远在你手里
- **统计面板** — 复盘数量、成功率、产出趋势
- **全文搜索** — 按关键词搜索日记、复盘、心得
- **离线写入** — 断网也能记碎片，联网自动同步

## 快速开始

### 环境要求

- Docker + Docker Compose
- Anthropic API Key

### 部署

```bash
# 克隆项目
git clone https://github.com/yourname/moleng.git
cd moleng

# 配置环境变量
cp .env.example .env
# 编辑 .env，填入你的 ANTHROPIC_API_KEY

# 一键启动
docker-compose up -d
```

启动后访问 `http://localhost` 即可使用。

### 本地开发

```bash
# 后端
cd backend
pip install -r requirements.txt
uvicorn app.main:app --reload

# 前端
cd frontend
npm install
npm run dev
```

## 数据结构

四张表，单向链条：

```
收集箱（碎片暂存）→ 日记（原始记录）→ 复盘（事件 + 归因 + 标签）→ 心得（知 + 行）
```

- **收集箱** — 碎片暂存区，每天整合进日记后清空
- **日记** — 一天一条，确认后不再修改，作为原始素材永久保留
- **复盘** — 从日记中提取的事件，包含决策、偏差、归因，带标签（成功/失败/认知迭代）
- **心得** — 从复盘中提炼的认知和 SOP，支持迭代链（新心得关联旧心得）

## 设计哲学

**AI 帮你发现值得思考的事，你自己思考，数据库帮你记住思考的结果。**

- 输入端零门槛 — 什么都能往里扔，不用想"这条值不值得记"
- AI 只做辅助 — 提取事件、给参考建议，不替你思考
- 输出端你把关 — 复盘和心得全部由你亲手写，AI 的建议只是参考
- 空白日允许存在 — 没东西可写就不写，不强迫每天产出
- 数据在你手里 — 自部署，SQLite 文件在你的服务器上，随时可导出
